# Quick Reference: Teacher Editing

## How to Edit Teachers

### Simple Fields (Inline Editing)

**Editable Fields:**
1. Max Periods per Week
2. Time Preference (Morning/Afternoon/None)
3. Max Periods per Day
4. Max Consecutive Periods

**How to Edit:**
1. Click on any field value
2. Edit the value (number or select dropdown)
3. Press **Enter** to save or **Esc** to cancel
4. UI updates immediately after save ‚úÖ

### Complex Fields (Expandable Row)

**Editable:**
1. Weekly Availability (Calendar Grid)
2. Primary Subjects
3. Allowed Subjects
4. Subject Restrictions

**How to Edit:**
1. Click the **chevron icon** (‚ñº) to expand the row
2. Choose tab: **Availability** or **Subjects**
3. Make your changes
4. Click **Save Changes** (for availability)
5. Changes apply immediately for subjects
6. Press **Esc** or click chevron again to collapse

## Common Actions

### Add Teacher
1. Click **Add Single** button
2. Fill in the form
3. Click **Save Teacher**

### Edit Teacher Field
1. Click the field value
2. Change it
3. Press Enter

### Edit Availability
1. Click chevron to expand row
2. Go to **Availability** tab
3. Click periods to toggle
4. Click **Save Changes**

### Edit Subjects
1. Click chevron to expand row
2. Go to **Subjects** tab
3. Click subject badges to toggle
4. Changes save automatically

### Delete Teacher
1. Click **trash icon** on the right
2. Confirm deletion

### Duplicate Teacher
1. Click **copy icon** on the right
2. Teacher copied with " (Copy)" suffix

### Bulk Delete
1. Check boxes next to teachers
2. Click **Delete Selected**
3. Confirm deletion

## Troubleshooting

### UI not updating?
- **Fix**: It should update automatically now with the new architecture
- Check console for errors
- Refresh page if needed

### Can't edit a field?
- Make sure you're clicking the value, not the label
- Look for the pencil icon on hover

### Changes not saving?
- Check for error toast messages
- Check browser console
- Verify internet connection
