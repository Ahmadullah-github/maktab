# Font Assets

This directory contains font files for PDF generation.

## Vazirmatn Font

The Vazirmatn font should be placed here for Persian text support in PDF
exports.

Required files:

- `Vazirmatn-Regular.woff2` - Main font file for Persian text

## Font Licensing

Ensure proper licensing for any fonts used in the application. Vazirmatn is
available under SIL Open Font License.

## Usage

The PDFGenerationService will automatically load fonts from this directory and
embed them in generated PDF files for proper Persian text rendering.
